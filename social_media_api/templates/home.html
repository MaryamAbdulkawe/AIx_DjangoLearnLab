Quick Start

1. Register at /api/accounts/register/

2. Login at /api/accounts/login/

3. Create posts at /api/posts/

4. Explore feed at /api/posts/feed/

Features

  • User Authentication
  • Post Management
  • Comment System
  • Like/Unlike Posts
  • Follow/Unfollow Users
  • Personalized Feed
  • Token Authentication
  • Admin Interface
